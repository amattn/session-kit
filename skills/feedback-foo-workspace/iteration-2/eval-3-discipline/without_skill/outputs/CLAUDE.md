## Rename Discipline

**After every file or variable rename, complete these steps before moving on:**

1. Run a consistency pass — search the codebase for all references to the old name
2. Update every stale reference found
3. Update NOTES.md to record what was renamed and why
4. Do a final check for any remaining stale references

Do not proceed to the next task until all four steps are complete. Skipping this has caused bugs in sessions 2, 4, and 5.
